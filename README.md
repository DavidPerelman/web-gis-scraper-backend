# תוכנית פרויקט: Web GIS Scraper

## תיאור כללי

פרויקט זה נועד לאפשר למשתמשים להעלות קובץ פוליגון גיאוגרפי (כגון קובץ Shapefile או GeoJSON), ולבצע עליו עיבוד בצד שרת הכולל גרידת נתונים מאתר תכניות בניה ממשלתי. לאחר העיבוד, המשתמש יקבל קובץ חדש המכיל את הפוליגון בצירוף הנתונים הרלוונטיים לכל תכנית שנמצאה בתוכו.

## מטרות

- לאפשר קבלת מידע על תכניות בניה באמצעות כלי אינטרנטי.
- להדגים יכולות GIS, scraping, ו־Full Stack.
- לשפר את מבנה הקוד והתיעוד לפרויקט קודם שנכתב לצורך פנימי.

## טכנולוגיות

### Backend

- Python 3.11+
- FastAPI
- GeoPandas
- Shapely
- Selenium / BeautifulSoup (גרידה)
- pandas

### Frontend

- Next.js (React)
- Leaflet.js (תצוגת מפה)

### כלים נוספים

- Docker (לשלב בהמשך)
- GitHub Actions (CI/CD בעתיד)

## מבנה תיקיות מוצע


## שלבי עבודה

1. **איפיון MVP**:

   - העלאת קובץ פוליגון
   - גרידת נתונים עבור הפוליגון
   - יצירת שכבת פלט והורדתה

2. **הקמת צד שרת**:

   - יצירת API לקבלת קובץ Shapefile/GeoJSON
   - המרת הפוליגון
   - חיבור ללוגיקת גרידה קיימת
   - יצירת קובץ GeoJSON/Excel להורדה

3. **הקמת צד לקוח**:

   - עמוד העלאה והצגת מפה
   - שליחה לשרת, קבלת תוצאה, תצוגה וכפתור הורדה

4. **שיפור UI/UX + תיעוד**:

   - עיצוב נקי ופרקטי
   - README מקצועי + סרטון דמו (בהמשך)

5. **דיפלוימנט והצגה בפורטפוליו**:
   - פריסה ב־Render/Netlify/Verce או Docker מקומי
